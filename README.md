# Vibe Coding Learning

一个围绕“简历优化微信小程序”展开的学习型项目仓库，用于沉淀产品思考、交互原型和需求文档，并为后续前端开发或小程序实现提供基础资料。

## 项目简介

这个仓库当前聚焦在产品方案验证阶段，核心目标是围绕具体岗位 JD，对用户现有简历进行定向优化，帮助提升简历与岗位要求的匹配度，以及 ATS / 初筛通过率。

目前仓库内容以原型和文档为主，适合用于：

- 梳理产品方向和页面流程
- 展示简历优化小程序的交互思路
- 为后续前端开发、小程序开发提供参考基础

## 仓库结构

```text
VibeCodingLearning
├── index.html
├── README.md
├── 简历优化小程序prd.md
└── 前端UI交互稿/
    └── README.md
```

## 文件说明

- `index.html`
  简历优化小程序的交互原型页面，可直接在浏览器中打开查看。

- `简历优化小程序prd.md`
  项目的产品需求文档，包含产品定位、目标用户、核心流程、页面结构和功能需求。

- `前端UI交互稿/`
  用于存放从 Figma 或其他设计工具导出的交互稿、说明文件和相关资源。

## 当前阶段

当前项目仍处于原型沉淀阶段，重点放在以下几个方面：

- 明确 MVP 范围
- 梳理页面结构与核心流程
- 搭建可演示的交互原型
- 为后续开发做资料准备

## 如何预览

### 方式一：直接打开

在本地浏览器中直接打开 `index.html` 即可查看当前原型。

### 方式二：使用本地静态服务

如果你希望用本地服务方式预览，可以在项目目录运行：

```bash
python3 -m http.server 8000
```

然后在浏览器访问：

```text
http://localhost:8000
```

## 项目方向

围绕“1 个 JD + 1 份简历 = 1 次定向优化任务”的核心单位，逐步完善以下能力：

- 岗位 JD 输入与解析
- 原始简历上传与管理
- 定向诊断与优化建议展示
- 优化前后效果对比
- 简历资产管理与历史记录查看

## 后续计划

- 视需要扩展为可运行的小程序或 Web 演示版本
