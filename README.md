# 简历优化微信小程序 UI

这是一个基于 PRD 文档构建的简历优化微信小程序 UI 演示。使用 React + Tailwind CSS + React Router 开发。

**版本更新：** 已根据 PRD V1.0 最新版本更新UI（第三次迭代）

## 最新更新内容（第三版）

- ✅ JD输入方式：改为"文本粘贴"和"上传JD截图"
- ✅ 简历输入方式重构：
  - 第一步：选择"上传新简历"或"已有简历"
  - 上传新简历支持：文本粘贴 / 上传Word
  - 已有简历分类展示：原始简历在前，优化后简历在后

## 之前更新内容

- ✅ 首页最近优化记录显示优化前后匹配度对比
- ✅ 优化案例改为列表形式，展示5个案例
- ✅ 新增积分不足引导弹窗
- ✅ 支持观看视频广告获取积分
- ✅ 更新原始简历格式标签（Word/文本）

## 功能特性

### 核心流程
- **首页** - 简历优化入口、最近优化记录、优化案例展示
- **简历提交页** - 支持多种 JD 和简历输入方式、AI 模型选择
- **诊断动画页** - AI 分析进度展示
- **诊断报告页** - 匹配度分析、JD 解读、优化建议
- **补充信息页** - 聊天式/问题清单两种补充模式
- **优化结果页** - 完整预览、差异对比
- **格式对齐页** - 还原原始简历格式

### 简历资产管理
- **简历列表页** - 原始简历和优化记录管理
- **简历查看页** - 优化前后三视图对比
- **任务详情页** - 完整优化信息展示
- **原始简历查看页** - 查看和导出原始简历

### 用户中心
- **我的页面** - 会员状态、积分余额、功能入口
- **会员/积分充值页** - 支持会员开通和积分充值
- **订单记录页** - 查看充值购买订单
- **积分明细页** - 积分变动记录
- **问题反馈页** - 用户反馈收集

## 设计特色

### 微信小程序风格
- 采用微信小程序标准绿色主题色 (#07C160)
- 模拟小程序顶部状态栏
- 底部导航栏符合小程序规范
- 支持安全区域适配

### 交互设计
- 卡片式布局，清晰直观
- 渐进式信息展示
- 流畅的页面跳转动画
- 移动端优化的触控体验

### 功能亮点
- 多种简历和 JD 输入方式
- AI 模型可选择
- 聊天式信息补充
- 优化前后对比展示
- 完整的积分和会员体系

## 页面路由

- `/` - 首页
- `/submit` - 简历提交
- `/diagnosis` - 诊断中
- `/report` - 诊断报告
- `/supplement` - 补充信息
- `/result` - 优化结果
- `/format` - 格式对齐
- `/resumes` - 简历列表
- `/resume/original/:id` - 原始简历查看
- `/resume/view/:id` - 简历查看
- `/task/:id` - 任务详情
- `/profile` - 我的
- `/payment` - 充值中心
- `/orders` - 订单记录
- `/points-history` - 积分明细
- `/feedback` - 问题反馈

## 技术栈

- React 18
- React Router 7
- Tailwind CSS v4
- Lucide React (图标)
- TypeScript

## 开发说明

本项目在 Figma Make 环境中运行，开发服务器已自动启动。所有页面均已实现响应式布局，适配移动端显示。
