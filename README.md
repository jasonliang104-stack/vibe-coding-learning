# Vibe Coding Learning

这是一个围绕“简历优化微信小程序”整理的学习型项目仓库，当前主要用于沉淀产品想法、前端交互原型和相关资料。

## 仓库内容

- `index.html`：简历优化小程序交互原型页面
- `简历优化小程序prd.md`：产品需求文档（PRD）
- `前端UI交互稿/`：前端 UI 交互稿与说明文件

## 当前阶段

当前仓库以原型和文档为主，适合用于：

- 梳理产品方案
- 展示页面交互思路
- 作为后续小程序开发的基础资料

## 如何查看原型

可直接在浏览器中打开 `index.html` 进行预览。

- 持续完善 PRD
